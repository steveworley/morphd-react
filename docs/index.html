/blocks get

get

Get a list of blocks.

get /blocks

Get a list of blocks.

  • Request
  • Response

Query Parameters

  • limit: (number)

    Determine how many results to return

  • p: (number)

    Determine where to start the limit from

HTTP status code 200

Body

Type: application/json

Schema:

{
  "$schema": "http://json-schema.org/draft-03/schema",
  "id": "http://jsonschema.net",
  "required": true,
  "properties": {
    "delta": {
      "type": "string",
      "required": true
    },
    "bid": {
      "type": "number",
      "required": true
    },
    "title": {
      "type": "string",
      "required": true
    },
    "content": {
      "type": "string",
      "required": true
    },
    "weight": {
      "type": "number"
    }
  }
}

Example:

  },

      "bid": 1,

    }
